# 의존성 주입(Dependency Injection, DI)

객체가 스스로 의존 객체를 생성하지 않고, 객체 간의 의존 관계를 외부에서 주입하는 외부에서 주입 받는 패턴

외존성 주입을 사용하면 코드 모듈성 향상 및 테스트에 용이하고, 유지보수성을 높일 수 있다.


# 용어

- 의존성(Dependency) : 한 클래스가 다른 클래스에 의존하는 관계를 말함(어떤 클래스 A가 다른 클래스 B를 사용하여 동작할 때, A는 B에 의존한다고 말함).

- 주입(Injection) : 필요한 의존성을 클래스가 직접 만들지 않고 외부에서 주입 받아 제공받는 방식을 말함.

- DI 컨테이너(DI Container) : 일반적으로 의존성 주입을 관리하는 주체에 해당하는 프레임워크의 컴포넌트를 말함(지시하면 객체 생성과 의존성 연결을 알아서 진행함).
